About the Site

Welcome to Anime Made Easy! The purpose of this website is provide anime fans and newcomers to the medium with an online environment where they can learn more about anime. The hope is that this website will help alleviate some of the nervousness that comes with first discovering something new and eliminate any mainstream misconceptions that prevent people from fully exploring the medium.

How it's built

With the help of Professor Mark Rayner's instructional class website and the W3Schools tutorials I was able to code everything for this site by hand by using a program called Brackets. It is built using HTML5 and CSS which are standards of web design.

As well, by using image manipulation programs like Photoshop and Krita, I made all the backgrounds for each page by cutting out transparents of anime characters and then layering them on top of one another to make several unique collages that fit with the theme of the page. A similar method was used to create the Anime Made Easy banner that appears on each page.

Font

The typesetting for this website is done in Cinzel and Oswald, both of which are available on Google Fonts. Cinzel is an easy to read serif typeface and Oswald is a sans-serif that nicely pairs with Cinzel.
